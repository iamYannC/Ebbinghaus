## Project overview

An R-based benchmark for testing how AI vision-language models perceive the Ebbinghaus illusion. Three-phase pipeline: stimulus creation, model evaluation, analysis.

## Working directory

All `source()` calls and file paths are relative to the project root (`Ebbinghaus/`). Set your working directory there before running anything.

## Data schemas

### trials.csv (32 columns)
- **ID:** `trial_id`, `n_comparisons` (fixed at 2)
- **Layout:** `orientation`, `canvas_width`, `canvas_height`, `background_color`
- **Test stimuli (A and B):** `test_{a,b}_{shape,size,color,fill}`
- **Surround stimuli (A and B):** `surround_{a,b}_{shape,size,n,color,fill,distance}`
- **Ground truth:** `true_larger`, `true_diff_ratio` (decimal, not pct), `tier` (0/1/2/3/NA)
- **Reproducibility:** `seed`
- **Rendering:** `file_format`, `file_path`

### Key conventions
- `true_diff_ratio` is a decimal proportion `[0, 1)`, not a percentage.
- `tier` parameter in `generate_trial()`: `NULL` = unconstrained, `0`/`1`/`2`/`3` = targeted. `NA` is only an output of `classify_tier()`, never a valid input.
- `generate_design()` is a pure function — returns the data frame without writing to disk.
- Seed auto-generation uses `round(runif(1, -1e9, 1e9))`, not `sample(-1e9:1e9, 1)` (the latter materializes a 2B-element vector).
